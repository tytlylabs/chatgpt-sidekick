# Privacy Policy for ChatGPT SideKick

**Effective Date**: November 1, 2025

## Data Collection
We do NOT collect, store, or transmit any personal data to external servers.

## Local Storage
- Premium license keys stored in Chrome sync storage
- No data sent to external servers except Gumroad API for license validation

## CSP Modification
This extension modifies Content-Security-Policy headers for:
- chatgpt.com
- chat.openai.com

This is ONLY for split-screen iframe functionality. We do NOT modify headers for any other websites.

## Third-Party Services
- Gumroad API: Used only for license validation when you activate premium features
